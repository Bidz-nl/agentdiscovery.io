# Security Policy

## Supported versions

The current repository should be treated as the actively maintained MVP line unless stated otherwise in release notes or repository documentation.

## Reporting a vulnerability

If you discover a security vulnerability, please report it responsibly and do not open a public issue first.

Please send a report to:

- `security@agentdiscovery.io`

Include, if possible:

- a description of the issue
- affected files or endpoints
- reproduction steps
- impact assessment
- any suggested mitigation

## Responsible disclosure

Please allow maintainers a reasonable amount of time to investigate, confirm, and address the issue before public disclosure.

We appreciate coordinated, responsible disclosure that helps protect users and contributors.
