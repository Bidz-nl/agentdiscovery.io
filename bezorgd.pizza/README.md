# bezorgd.pizza

Standalone branded pizza-ordering product built on a small reusable local-food domain layer and a minimal ADP/core foundation.

## Phase 1 priorities

- public product first
- provider workspace second
- ADP/core minimal
- warm, local, no-nonsense UX

## Initial scaffold

This repository currently contains:

- architecture and phase-1 docs
- root config for a Next.js app
- minimal public and workspace app shell files

Feature implementation is intentionally deferred until after the scaffold phase.
